IDs of conversations that shouldn't trigger a notification. To find the conversation ID, select the conversation (chat or channel) in teams.

  • For Channels: Copy the id between threadId= and @ in the address bar.
  • For Chats: Copy the id between _#conversations/ and @ in the address bar.

You probably want to put your own name in there. The name needs to be the same as is displayed in the notifications.
